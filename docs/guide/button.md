## Class Yiisoft\Yii\Bootstrap4\Button
Renders a [Bootstrap Button component](https://getbootstrap.com/docs/4.5/components/buttons/).

### Usage

```php
declare(strict_types=1);

use Yiisoft\Yii\Bootstrap4\Button;

echo Button::widget()
    ->label('Action')
    ->options(['class' => 'btn-lg']);
```
